## Getting Started With create-react-app

### Installation

Clone this repository:

```sh
git clone https://github.com/Kamahl19/react-starter my-app
cd my-app
```

Run the client

```sh
cd client
npm i
npm start
```

Run the backend

```sh
cd backend
npm i
npm start
```
