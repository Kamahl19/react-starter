## Getting Started With create-react-app

### Installation

Clone this repository:

```sh
git clone https://github.com/Kamahl19/react-starter my-app
cd my-app
```

Seed DB & Start the backend

```sh
cd backend
yarn install
yarn run seed
yarn start
```

Start the client

```sh
cd client
yarn install
yarn start
```
